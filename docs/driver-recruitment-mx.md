## Posting Strategy

**Day 1-3:** Post main message in 2-3 groups
**Day 4-7:** Reply to comments, answer questions
**Week 2:** If <5 drivers, post in remaining groups

**DO NOT:**
- Spam multiple posts per day
- Promise things not yet built
- Criticize other platforms directly
- Use fake accounts

**DO:**
- Respond to every comment
- Be transparent about being new
- Invite feedback openly
