## Monitoring Checklist (Post-Publish)

### Daily Checks
- [ ] App crashes / ANRs in Play Console
- [ ] Login or verification errors
- [ ] Onboarding completion rate
- [ ] Ride creation success rate
- [ ] Bot/notification delivery

### Issue Severity
- **Critical:** App crash, can't login, can't accept rides
- **Medium:** UI bugs, slow loading, notification delays
- **Minor:** Text typos, minor visual issues

## Soft Launch Criteria

DO NOT move to public until ALL are true:
- [ ] Onboarding works flawlessly
- [ ] 5-10 drivers completed real rides without errors
- [ ] Telegram/WhatsApp bots operating correctly
- [ ] Dispute handling tested and working
- [ ] No critical bugs for 72 hours
- [ ] Driver feedback collected and addressed
